About

Designer who
speaks both sides.

Based in Visakhapatnam, I work at the edge of design and code — fluent enough in both to move fast and ship things that actually work.

I've built design systems from scratch, run user research, and delivered polished UI for clients across India and internationally. I care about craft, clarity, and the details that most people skip.

Process
01
Understand first
Every project starts with research — user interviews, heuristic audits, or competitive benchmarking. I don't open Figma until I know the problem.
02
Structure the logic
Information architecture, user flows, content hierarchy. The invisible work that makes the visible work feel obvious.
03
Design with intent
High-fidelity UI built on a proper token foundation. Every decision is defensible — nothing is decorative without reason.
04
Ship and iterate
I write handoff-ready specs and can prototype in code. I stay engaged through implementation — not just until delivery.
